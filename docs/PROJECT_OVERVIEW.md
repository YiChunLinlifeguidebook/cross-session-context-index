# Nine Palace Index

Nine Palace Index is an experimental indexing and weighting project developed in Termux on mobile.

## Goals
- Build a structured indexing mechanism
- Explore weighted decision logic
- Keep the system lightweight and portable
- Develop directly from constrained mobile environments

## Current Status
Early prototype with C/C++ source files, configuration, and command-line workflow.

## Development Environment
- Device: OPPO Reno 14
- Terminal: Termux
- Version control: Git + GitHub
- Remote sync: SSH

This project is part of a broader creative and technical direction connected to:
