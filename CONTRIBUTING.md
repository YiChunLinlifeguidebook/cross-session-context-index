# Contributing

Thank you for your interest in contributing.

## Basic Flow

1. Fork this repository.
2. Create a feature branch.
3. Make your changes.
4. Run tests:

```bash
make test
```

5. Open a Pull Request describing your change.

## 歡迎的改進方向

- 中文分詞 / 更好的關鍵字比對邏輯
- 跨平台二進位相容性(目前直接讀寫 struct)
- 改用 SQLite 等更穩健的儲存方式
- 記錄的刪除 / 修改功能
- 單元測試
